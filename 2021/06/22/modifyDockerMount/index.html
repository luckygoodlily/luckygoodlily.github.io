修改docker預設mount的位置ount

當我們硬碟不夠時，會掛載新的硬碟進來，這時候就會需要把docker預設mount的位置改到大的disk中

  1. 停止 docker daemon
  • sudo service docker stop
  1. 加一個config檔案和告訴 docker daemon 檔案位置
    增加的檔案名稱為 daemon.json 在 /etc/docker目錄下

檔案內容如下:

也可修改/lib/systemd/system/docker.service:

  1. 複製所有檔案到新位置

sudo rsync -aP /var/lib/docker/ /path/to/your/docker

  1. 重新命名就檔案名稱

sudo mv /var/lib/docker /var/lib/docker.old

  1. 重起docker deamon

sudo systemctl restart docker
或是
sudo service docker start

  1. 測試

docker volume inspect tour_mount_name


如果Mountpoint改變成你設定的路徑就是成功了，如果一切內容都沒問題，就可以把原來的docker.old刪掉了

sudo rm -rf /var/lib/docker.old
