python 的list綜合表達式

使用for語法，達成list生成一系列值得目的，還能套用在dictionary和set集合物件上。

逐元素複製list所有元素

法一

b_list=a_list[:]

法二

由於法二的方法太常見了，python 2.0引入了一種緊湊的方法來做同樣的事情。
b_list = [i for i in a_list]

延伸寫法。假設建立一個list，內容為a_list中元素的平方：

如果a_list包含[1,2,3]，結果內容是[1,4,9]，對應的綜合表達式為：
b_list = [i*i for i in a_list]

規律為：

[value for_statement_header]

雙層迴圈

結果
=>[0,0,0,0,1,2,0,2,4]

綜合表達式

mult_list = [i*j for i in range(3) for j in range(3)]

綜合表達式if規律為：

[value for_statement_header if_expression]

上面程式執行結果，會在new_list中放置值[10,12,13,15]，下面的述句改為使用list綜合表達式執行相同的動作：
new_list = [i for i in my_list if i > 0]

list 綜合表達式中括號內的右側，在本例中它由三個部份組成：

  1. value 運算式i;直接從list中取一個值。
  2. for述句for i in my_list，功能是產生要用的數列。
  3. 最後，if 條件句，if i > 0，用來選擇哪些元素會參加運算。
