False Positives和False Negative含意

實際 Positive Cases 實際 Negative Cases
Predicted Positive True Positive O False Positive X （假陽性）
Predicted Negative False Negative O True Negative X

True Positive （真正, TP）被模型預測為正的正樣本；

True Negative（真負 , TN）被模型预測為的負樣本 ；

False Positive （假正, FP）被模型预測為正的負樣本；

False Negative（假負 , FN）被模型预測為负的正樣本；

True Positive Rate（真正率 , TPR）或靈敏度（sensitivity）
TPR = TP /（TP + FN）
正樣本預測结果數 / 正樣本實際數

True Negative Rate（真负率 , TNR）或特指度（specificity）
TNR = TN /（TN + FP）
負樣本预測结果數 / 負樣本實際數

False Positive Rate （假正率, FPR）
FPR = FP /（FP + TN）
被预測為正的負樣本结果數 /負樣本實際數

False Negative Rate（假负率 , FNR）
FNR = FN /（TP + FN）
被预測為負的正样本结果數 / 正樣本實際數
