爬蟲 Dcard API 2.0

最近在練習爬蟲，想說要來爬Dcard，但是觀察到XHL長的跟過去不太一樣，於是整理出來差異

公開API

Dcard API 1.0

https://dcard.tw/_api

Dcard API 2.0

https://www.dcard.tw/service/api/v2

功能 請求方式 路徑 回傳
全部文章 GET /posts Json
看板資訊 GET /forums Json
看板內文章列表 GET /forums Json
文章內文 GET /posts/{文章ID} Json
文章內引用連結 GET /posts/{文章ID}/links Json
文章內留言 GET /posts/{文章ID}/links Json

API參數介紹

popular

預設使用"最新"文章來排序，透過popular參數可以切換 “最新” 與 “熱門”，如下：
最新文章 -> https://www.dcard.tw/service/api/v2/posts?popular=false
熱門文章 -> https://www.dcard.tw/service/api/v2/posts?popular=true
個別熱門看板 -> https://www.dcard.tw/service/api/v2/forums/makeup/posts?popular=true&limit=100
預設為 popular=false

limit

預設回傳文章為"前30"筆文章，limit參數來增加回傳文章數量，最高限制是100筆
熱門文章前100筆 -> https://www.dcard.tw/service/api/v2/posts?popular=true&limit=100

before

有時會需要取得超過100筆文章，在前一次路徑後增加before，before加前一次路徑最後一筆文章的id
https://www.dcard.tw/service/api/v2/posts?popular=true&before=234267056
