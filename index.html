ELK

ELKCmd

docker-compose 裝Elk常用指令

查特定port

netstat -an|grep 5044

查正在監聽的port

netstat -plnt

監聽elaticsearch

curl "http://127.0.0.1:9200/_cat/nodes"
和
curl -XGET "http://127.0.0.1:9200/_cat/nodes"
是一樣的

查詢log

docker-compose logs

查詢特定container log

docker-compose logs logstash
或是
docker logs -t -f logstash

查詢machine

$ docker-machine ls

查詢mount的空間

$ docker volume ls

查詢設置得網路

$ docker network ls

查詢有哪些image

$ docker images

查詢啟了哪些container和port

$ docker ps
